# Карточная игра "Архив"

**«Архив»** — это *соревновательная карточная игра*, сочетающая механику покера и стратегического накопления ресурсов. В отличие от классических карточных игр, здесь побеждает не та рука, которая была роздана, а та, которую игрок собрал сам в течение партии.

Игрок получает основную руку для атак и защиты. Но победу определяет не она. В течение партии игрок собирает Архив — набор карт, который формируется за счёт успешных действий. В финале из Архива и общих карт строится лучшая покерная комбинация. Карты основной руки никогда не участвуют в определении победителя.

# Игровой цикл делится на:
1) Раздача — каждый игрок получает 4 карты.
2) Обязательная ставка — все игроки делают стартовый взнос.
3) Атака и защита — игроки по очереди атакуют картами. Соперник может защититься (побить старшей картой) или пропустить ход.
4) Архив — успешная атака или защита отправляет карту в Архив игрока.
5) Открытие общих карт — после определённого количества успешных защит открываются общие карты (сначала 3, потом 4-я, потом 5-я).
6) Торговля — после каждого открытия начинается новый раунд ставок.
7) Showdown — когда открыты все 5 общих карт, проводится финальное вскрытие. Победитель определяется по лучшей покерной комбинации из Архива + 5 общих карт.

# Ключевые механики игры: 

- Архив — личная коллекция карт игрока. Он растёт по ходу партии и определяет итоговую комбинацию.
- Основная рука — используется только для атак, защиты и тактических решений. В финале не учитывается.
- Общие карты — открываются постепенно, добавляя всем игрокам общие ресурсы для комбинаций.
- Торговля — позволяет блефовать, повышать ставки и выходить из игры.
- Покерные комбинации — определяются автоматически из Архива + общих карт по классическим правилам покера.

# Технологии, используемые при создании игры:

Клиент: HTML, CSS, JavaScript (Canvas)

# Как запустить игру:
# Установить зависимости (через cmd в папке с файлами игры)
npm init -y
npm install ws

# Запустить сервер
node server.js

# Открыть в браузере (localhost соответственно меняется на ip человека, к игре которого вы подключаетесь)
http://localhost:8080
